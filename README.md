# ü§ñ AI Chronicles: The Story of Intelligent Machines

**AI Chronicles** is a multi-page educational website designed to take users on a journey through the evolution of Artificial Intelligence. From the theoretical foundations laid by Alan Turing to the modern era of Deep Learning and the ethical questions surrounding AGI.

## üöÄ Project Overview

This project simulates an "AI Access Terminal" entry point that leads into a comprehensive resource about the past, present, and future of AI. It demonstrates front-end web development skills using a mix of custom CSS and modern frameworks.

**Developer:** Saad Al-Shargabi  
**Status:** Active / Completed

## üìÇ Site Structure & Features

The project consists of 6 core pages:

### 1. The Access Terminal (`1.html`)
- **Feature:** A simulated command-line/terminal interface login screen.
- **Functionality:** Accepts user credentials to "unlock" the site or allows a bypass to the database.

### 2. Home & History (`2.html`)
- **Feature:** Interactive Image Slider (WOWSlider) and timeline.
- **Content:** Covers the eras of AI, from the 1950 Turing Test to the Deep Learning revolution of the 2010s.

### 3. Real-World Applications (`3.html`)
- **Feature:** Detailed breakdown of modern AI usage.
- **Content:** Explores Computer Vision, Natural Language Processing (NLP), and Autonomous Robotics.

### 4. Ethics & Future (`4.html`)
- **Feature:** Critical analysis of AI challenges.
- **Content:** Discusses Algorithmic Bias, Accountability, and the concept of Artificial General Intelligence (AGI).

### 5. Pioneers (`5.html`)
- **Feature:** Grid layout of historical figures.
- **Content:** Biographies of Turing, McCarthy, Hinton, LeCun, and Bengio.

### 6. Contact (`contact.html`)
- **Feature:** Modern UI using **Bootstrap 5**.
- **Functionality:** Includes a modal popup form for sending messages and social media integration.
